# Profile Image Instructions

Please save your profile image as `profile.jpg` in this folder (`/public`).

## Steps:
1. Take the image you shared in the chat
2. Save it as `profile.jpg`
3. Place it in the `/public` folder (where this file is located)

The image will automatically appear on:
- Homepage (Hero section) - as a circular profile photo
- Resume page (Header section) - as a smaller circular profile photo

## Image Requirements:
- Filename: `profile.jpg` (or `profile.png`)
- Recommended size: At least 500x500 pixels
- Format: JPG, PNG, or WebP
- The image will be automatically cropped to a circle

Once you add the image, refresh your browser to see it!
